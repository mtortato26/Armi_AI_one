<h3>Pinout</h3>

|Arduino nano|A0 - A7 -- D0 - D13|
|---|---|
|Ethernet Shield Enc28j60|D10 - D13|
|Drive H L9110s|D2 - D5|
|Servo motor Sg90|D6|
|Display Oled SSD1306|A4 - A5|
|Buzzer|A7|
|infra-red Lm393|A0 - A3 - A6|
|Serial port|D0 - D1|
